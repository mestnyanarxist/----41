Клонирование репозитория:
git clone git@github.com:мой_логин/мой_проект.git
Список веток(branch):
git branch
Создание новой ветки из мастер ветки(branch):
git checkout -b имя-новой-ветки.
Переключение ветки:
git checkout имя_ветки
Изменение имени ветки:
git branch -m старое-имя-ветки новое-имя-ветки.
Удаление ветки:
git push origin --delete имя_ветки
Сохранение всего:
git add -A 
Коммит:
git commit -m "ваше сообщение" 
Отправка изменений в Git:
git push origin название-текущей-ветки 
Для слияния веток нужно прописать в терминале: 
git checkout main ; 
git pull ; 
git merge <название> 
Чтобы в Git применились изменения тоже, нужно закомитить.
